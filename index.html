Configurar sincronización

Para sincronizar entre tu computadora y celular, necesitás una clave gratuita de JSONBin.io. Solo se hace una vez.

  1. Entrá a jsonbin.io y creá una cuenta gratuita
  2. Andá a API Keys en el menú
  3. Copiá tu Master Key (empieza con $2b$)
  4. Pegala acá abajo y hacé clic en "Guardar y conectar"
